# Мой проект для Ассаймента 6

## Описание
This project demonstrates the implementation of saving user preferences using local storage, including:

User login and logout system.
Light and dark theme switching with the selected theme being saved.
Filtering elements with the filter state being preserved.

## Как запустить проект
1. Clone the repository or download the project files.
2.Open home.html or Destinations.html in your browser.
3.Ensure that scripts.js and styles.css are correctly linked.

## Основные функции
Authentication: Users can log in and out. User data is saved in local storage.
Theme Switching: Users can toggle between light and dark themes. The choice is saved in local storage.
Filtering: Users can filter a list of items, and the current filters are preserved.

## Технологии
HTML
CSS
JavaScript
Local Storage

## Структура проекта
home.html / Destinations.html: Main HTML files.
styles.css: Styles for the project.
scripts.js: Main JavaScript code for handling authentication, themes, and filtering.

## Как использовать
Enter your username and click "Login" to log in.
Use the button to toggle themes.
To filter items, enter text in the filter field.
